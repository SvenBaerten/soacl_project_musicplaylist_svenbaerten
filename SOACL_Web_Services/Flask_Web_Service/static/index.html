Python Flask Web Service for Project SOACL by Sven Baerten

This web service uses the Spotify REST API (see https://developer.spotify.com/documentation/web-api/).

Functions:

  • Get information about a music track via the artist name and the song title.
    • URL: /api/track/[artist]/[title]
    • Method: GET
    • URL parameters:
      • artist: the artist of the music track
      • title: the title of the music track
    • Headers:
      • Content-Type: application/json
    • Body parameters:
      • None
    • Success response:
      • Code 200 (OK) with response a JSON object with track information like in 'Sample call - Response'
    • Error response:
      • Code 500 (Internal Server Error)
    • Sample call:
      • Command:

        curl -X GET -H "Content-Type: application/json" "http://flask-web-service-svenbaerten.westeurope.azurecontainer.io/api/track/kanye%20west/stronger"
