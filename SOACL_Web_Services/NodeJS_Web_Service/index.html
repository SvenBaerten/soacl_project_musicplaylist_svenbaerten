Node.js Web Service for Project SOACL by Sven Baerten

Functions:

  • Split a date into the year, the month (with name) and the day.
    • URL: /api/dateSplitter/[date]
    • Method: GET
    • URL parameters:
      • date: the date of format year-month-day
    • Headers:
      • None
    • Body parameters:
      • None
    • Success Response:
      • Code 200 (OK) with response a JSON object with date information.
    • Error Response:
      • Code 500 (Internal Server Error)
    • Sample Call:
      • Command:

        curl -X GET -H "Content-Type: application/json" "http://nodejs-web-service-svenbaerten.westeurope.azurecontainer.io/api/dateSplitter/2019-8-21"

      • Response: 
                                            {
                                        "year":2019,
                                        "month":8,
                                        "monthNameLong":"August",
                                        "monthNameShort":"Aug.",
                                        "day":21
                                     }
