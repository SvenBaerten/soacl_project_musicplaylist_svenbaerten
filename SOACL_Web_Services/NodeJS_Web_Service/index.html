Node.js Web Service for Project SOACL by Sven Baerten

Functions:

  • GET /api/dateSplitter/[date]: split a date with format 'year-month-day' into year, month and day.
    Example: date 2019-2-22 gives {"year": "2019", "month": "2", "monthNameLong": "February", "monthNameShort": "Feb.", "day": "22"}.
